Урок 11 - Трансформації

2D Трансформація

  • translate(x, y)

    • -20px, -40px
    • 10px, 20px
    • 50%, 25%
    • 0, -100%

  • scale(x ,y)

    • 0.5
    • 110%
    • 50%, 25%
    • 50%, 1

  • rotate(x)

    • 45deg
    • 360deg
    • -10deg
    • 135deg

  • skew(x, y)

    • -15deg
    • 0, 15deg
    • 15deg, 15deg
    • 45deg, -45deg

  • matrix (multiple)- спробуй уявити що буде з цим блоком, якщо він..

    • .. має усі трансформації червоного кольору
    • .. має усі трансформації зеленого кольору
    • .. має усі трансформації синього кольору
    • .. має усі трансформації рожевого кольору

  • transform-origin: ___ ___;

    • 0 0
    • right bottom
    • center 50%
    • -100% -100%

3D Трансформація

  • translate3d(x, y, z)

    • -20px, -40px, 30px
    • 10px, 20px, -130px
    • 50%, 25%, 10px
    • 0, -100%, 0

  • scale3d(x, y, z)

    • 0.5, 0.5, 1
    • 110%, 2, 1
    • 50%, 25%, 2
    • 50%, 1, 0.5

  • rotate3d(x, y, z, deg)

    • 1, 1, 1, 45deg
    • 1, 1, 1, 360deg
    • 0, 0.5, 1, -10deg
    • 1, 1, 1, 135deg
